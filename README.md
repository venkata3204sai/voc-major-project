A minimal, responsive To-Do web application built with plain HTML, CSS and JavaScript.  
Tasks are stored in the browser using `localStorage`, so they persist between visits. The UI supports adding tasks with due date & time, listing tasks and deleting tasks.

## Features
- Add tasks with **title**, **due date**, and **due time**  
- Delete tasks (per-item)  
- Tasks persist via **`localStorage`**  
- Responsive UI using CSS (mobile-friendly layout)  
- Clean, minimal UI with animated interactions

- HTML5
- CSS3 (modern variables/flexbox)
- Vanilla JavaScript (ES6)
- Browser `localStorage` for persistence

## Run locally
No build tools required. To run the app:

1. Clone the repo
```bash
git clone https://github.com/venkata3204sai/voc-major-project.git
cd voc-major-project
```
2. Open index.html in your browser (double-click or via a small static server)
```bash
# optional: serve with Python (for CORS-free testing)
python -m http.server 8000
# then open http://localhost:8000
```
